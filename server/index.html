サーバー情報

PORUBU.com、hobymour.netドメインで提供されている一部のサービスは埼玉の自宅内のサーバーで稼働しています。

私の運営しているサービスの大半を稼働している自宅サーバーは、OCN配下で、不定期で変わる動的IPアドレスで稼働しています。

主に自作マシンで本体を動かしており、基本は自分専用ですが、一部のサービスは公開する予定です。

セルフホストAIに関してはさすがに用意できないので自分のPCでやってください…
